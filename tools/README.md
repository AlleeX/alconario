# Tools

Helper scripts for converting / packing assets.

Suggestions:

- **png2chr** — convert PNG sprite sheets to NES `.chr` (e.g. `nesst`, `nexxt`).
- **ftm2asm** — export FamiTracker `.ftm` to famitone2 `.s`.
- A small Python script that packs `levels/*.tmx` (Tiled) into compact
  nametable + collision binaries.

Add them here and call from the `Makefile` as a pre-build step.
